# 0 A.D.

A free, open-source, cross-platform real-time strategy game of ancient warfare. Currently in active development. Features multiplayer gameplay via a lobby server, historical civilizations, and full moddability. Multi-licensed (MIT / GPL-2.0 / Zlib). Source: <https://gitea.wildfiregames.com/0ad/0ad>

## Compatible Combos

| Infra | Runtime | Notes |
|---|---|---|
| Linux (Debian/Ubuntu) | deb package (ppa:wfg/0ad) | Easiest install path |
| Linux (any) | Build from source | Requires full C++ toolchain |
| macOS | .dmg download | Pre-built binaries from play0ad.com |
| Windows | .exe installer | Pre-built binaries from play0ad.com |
| Docker (dedicated server) | Custom | Community Dockerfiles exist for the multiplayer lobby |

> 0 A.D. is a game, not a web service — "self-hosting" here means running your own multiplayer lobby server.

## Software-Layer Concerns

- **Multiplayer lobby**: The official multiplayer lobby is XMPP-based (Jabber). Self-hosting a lobby requires running an XMPP server (e.g., ejabberd or Prosody) with 0 A.D. configuration.
- **Build dependencies**: Full source build requires CMake, Boost, SDL2, OpenGL, and many other libraries — see upstream build instructions.
- **Moddability**: Mods are placed in `~/.local/share/0ad/mods/` (Linux) or equivalent.
- **Data size**: Full install is large (1–2 GB) due to game assets.

## Deployment

### Option A — PPA (Ubuntu/Debian)

```bash
# Add Wildfire Games PPA
add-apt-repository ppa:wfg/0ad
apt update
apt install 0ad
```

### Option B — Distribution packages

```bash
# Arch Linux (AUR)
yay -S 0ad

# Fedora
dnf install 0ad
```

### Option C — Build from source

```bash
# Install dependencies (Debian/Ubuntu)
apt install build-essential cmake libboost-dev libsdl2-dev libpng-dev libxml2-dev

# Clone and build
git clone https://gitea.wildfiregames.com/0ad/0ad.git && cd 0ad
./build/workspaces/update-workspaces.sh
cd build/workspaces/gcc/ && make -j$(nproc)
```

### Multiplayer lobby server

See upstream wiki for XMPP lobby server configuration:
https://trac.wildfiregames.com/wiki/Multiplayer_Lobby

## Upgrade Procedure

- **PPA**: `apt update && apt upgrade 0ad`
- **Source**: `git pull`, then rebuild with `make -j$(nproc)` in the build workspace

## Gotchas

- **GPU requirements**: Requires OpenGL 2.1+ — software rendering is not supported. Headless servers cannot run the game client.
- **Lobby ≠ game server**: The XMPP lobby is just matchmaking. Actual game sessions are peer-to-peer by default.
- **In-development**: Game is feature-incomplete — some mechanics are still being implemented.
- **Large data**: Assets (~1–2 GB) are downloaded separately from the source code.
- **Build time**: Full source build can take 30–60 minutes on modest hardware.
