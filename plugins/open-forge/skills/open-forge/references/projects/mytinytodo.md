# myTinyTodo

**Simple AJAX-style todo list manager** written in PHP. Supports multiple lists, tags, priorities, due dates, and search. Works with SQLite (no database server needed), MySQL/MariaDB, or PostgreSQL. GTD-compliant workflow.

**Official site:** https://www.mytinytodo.net  
**Source:** https://github.com/maxpozdeev/mytinytodo  
**Demo:** https://www.mytinytodo.net/demo/  
**License:** GPL-2.0

## Compatible Combos

| Infra | Runtime | Notes |
|-------|---------|-------|
| Any | PHP + SQLite | Simplest — no separate DB needed |
| Any | PHP + MySQL/MariaDB | For shared/multi-user setups |
| Any | PHP + PostgreSQL | Supported |
| Any | Docker (PHP-FPM + nginx) | No official image; use generic PHP |

---

## System Requirements

- PHP 7.2+
- PHP extensions: `mbstring` + one of:
  - `pdo_sqlite`, `intl` — for SQLite
  - `pdo_mysql` or `mysqli` — for MySQL/MariaDB
  - `pdo_pgsql` — for PostgreSQL
- Web server (Apache with `mod_rewrite`, or nginx)

## Software-layer Concerns

### Installation
```bash
# Download latest release from https://www.mytinytodo.net/download.php
unzip mytinytodo-*.zip -d /var/www/mytinytodo
chown -R www-data:www-data /var/www/mytinytodo
# Browse to http://your-domain/mytinytodo/ and run the web installer
```

### Docker (PHP + nginx)
No official image; use a generic PHP container:
```yaml
services:
  app:
    image: php:8.2-apache
    volumes:
      - ./mytinytodo:/var/www/html
    ports:
      - '80:80'
```
Enable required PHP extensions in a custom Dockerfile if needed:
```dockerfile
FROM php:8.2-apache
RUN docker-php-ext-install pdo_mysql mbstring intl
```

### Config paths
| Path | Purpose |
|------|---------|
| `db/` | SQLite database files (back this up) |
| `config.php` | Generated by web installer |

### Apache `.htaccess`
myTinyTodo ships with an `.htaccess` for Apache clean URLs. Ensure `mod_rewrite` is enabled.

### nginx
```nginx
location / {
    try_files $uri $uri/ /index.php?$args;
}
```

## Upgrade Procedure

1. Back up the `db/` directory (SQLite) or dump your database
2. Download the new release
3. Extract, overwriting all files **except** `db/` and `config.php`
4. Visit the app — it will auto-apply any needed database schema upgrades

---

## Gotchas

- **SQLite is easiest** — no database server required; just back up the `db/` folder.
- **Write permissions** — the web server user must be able to write to `db/`. Set `chmod 755 db/` or `chown www-data db/`.
- **Single user by design.** myTinyTodo is a personal task manager; multi-user setups are not natively supported.
- **Password is optional** but recommended if accessible from the internet.
- **GTD-compliant:** supports inbox, project lists, next actions, and contexts via tags.
