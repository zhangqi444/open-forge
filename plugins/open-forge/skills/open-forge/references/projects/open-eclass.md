# Open eClass

**Advanced e-learning / Course Management System (CMS)** — complete platform for asynchronous e-learning used widely across Greek academic institutions. Supports course management, educational material storage/presentation, quizzes, assignments, video conferencing, and more.

**Official site:** https://www.openeclass.org  
**Source:** https://github.com/gunet/openeclass  
**Docs:** http://docs.openeclass.org/en/start  
**Demo:** https://demo.openeclass.org/  
**License:** GPL-2.0

## Compatible Combos

| Infra | Runtime | Notes |
|-------|---------|-------|
| Linux | PHP 8.1+ + MySQL/MariaDB + Apache/nginx | Primary install method |
| Any | Docker Compose | Community setups; no official Compose file in main repo |

## System Requirements

- PHP ≥ 8.1
- MySQL or MariaDB
- Web server (Apache recommended; nginx supported)
- Writable directories for uploads and course files

## Software-layer Concerns

### Installation
1. Download the latest release from https://www.openeclass.org/download (or clone the repo)
2. Extract to your web server docroot
3. Set write permissions on `courses/`, `storage/`, and `archive/` directories
4. Browse to your domain and follow the web installer
5. Complete the installation wizard (database connection, admin account, platform settings)

```bash
git clone https://github.com/gunet/openeclass.git /var/www/openeclass
chown -R www-data:www-data /var/www/openeclass
# Browse to http://your-domain/openeclass to run installer
```

### Key directories
| Path | Purpose |
|------|---------|
| `courses/` | Course content and uploaded files |
| `storage/` | User uploads and media |
| `config/config.php` | Generated by web installer; DB credentials |

### Features
- Course creation and management
- Educational materials (documents, video, links, exercises)
- Assignments with submission and grading
- Quizzes and tests
- Discussion forums
- Video conferencing integration
- Wiki and blog per course
- User enrollment with roles (admin, teacher, student, guest)
- Analytics and reporting
- Multilingual interface

---

## Upgrade Procedure

1. Back up database and `courses/` / `storage/` directories
2. Download and extract the new release over the existing installation
3. Browse to `http://your-domain/openeclass/upgrade/` and follow the upgrade wizard
4. Run database migrations via the wizard

---

## Gotchas

- **Greek academic origin.** UI and documentation are bilingual (Greek/English). The platform is production-tested across hundreds of Greek universities.
- **Web installer required.** No CLI install; must complete the browser-based setup wizard.
- **Write permissions critical.** The web server user must write to `courses/` and `storage/`. Incorrect permissions cause silent failures on file uploads.
- **Active development.** 228 commits in March 2026 alone — check the changelog before upgrading.
- **Demo available** at https://demo.openeclass.org/ for evaluation.
