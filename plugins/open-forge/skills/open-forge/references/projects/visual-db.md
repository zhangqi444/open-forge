# Visual DB

**What it is:** A web interface for databases — a visual GUI for browsing, querying, and editing database contents. Supports multiple database types. Closed-source product with a self-hosted tier.

> ⚠️ **Closed source.** No public repository. Self-hosting documentation at https://visualdb.com/self-host

**Official URL:** https://visualdb.com/self-host
**License:** Proprietary / closed source
**Stack:** Not publicly disclosed

---

## Compatible Combos

See upstream self-hosting documentation at https://visualdb.com/self-host for current deployment instructions and supported database types.

## Gotchas

- **Closed source** — no source code; dependent on vendor for updates and security
- **Open alternatives:** [Adminer](https://github.com/vrana/adminer), [Beekeeper Studio](https://github.com/beekeeper-studio/beekeeper-studio), [Baserow](https://gitlab.com/baserow/baserow), and [NocoDB](https://github.com/nocodb/nocodb) are popular open-source database GUI alternatives
