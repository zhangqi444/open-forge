# Webcap

**What it is:** A web archiving and change detection service. Monitor URLs for content changes, capture snapshots/screenshots, and track page history over time. Useful for monitoring competitor pages, tracking price changes, or archiving important web content.

> ⚠️ **Closed source.** No public repository. Self-hosting documentation available at https://rubyup.dev/webcap

**Official URL:** https://rubyup.dev/webcap
**License:** Proprietary / closed source
**Stack:** Not publicly disclosed
